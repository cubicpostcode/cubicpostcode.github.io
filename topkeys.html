Use the keys:

"Q" to move mousepointer cursor UP

"A" to move mousepointer cursor DOWN

"O" to move mousepointer cursor LEFT

"P" to move mousepointer cursor RIGHT

Press F11 to enter Full Screen mode

Work in progress for an open source free app for Linux with left and right clicking and dragging.

Will be easy to install and start from the terminal: with CTRL+ALT+T. And you will be able to kill it with CTRL+C or CTRL+Q.
