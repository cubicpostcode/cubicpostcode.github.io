Use the keys:

"Q" to move mousepointer cursor UP

"A" to move mousepointer cursor DOWN

"O" to move mousepointer cursor LEFT

"P" to move mousepointer cursor RIGHT

Press F11 to enter Full Screen mode

Work in progress for an open source free app for Linux, Windows and Mac,
with all mouse functionalities emulated, like left and right clicking and dragging.

Notice that two keys (vertical+horizontal) do produce diagonal movement.

Moreover notice that when it goes out of one side it comes from the other side of the screen.
Also check the speed. It will get better integrating it natively with the operating system as an interface enhancement.

The code will probably be in Python3, with the help of the module: https://pypi.org/project/PyAutoGUI/
