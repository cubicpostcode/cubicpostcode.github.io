Uma identidade que nunca se repete

"Sempre nunca igual" é mais do que um lema. É uma declaração de princípio: em cada momento, em cada programa, em cada nota, a Renascença se renova, surpreende e mantém-se fiel à sua essência sem nunca ser a mesma.

"Sempre nunca igual"

LINGUAGEM FORMAL

Sempre nunca igual

Expresso em notação matemática e lógica

1

Versão Principal

∀t (P(t) ≠ P(t′))

Para todo instante t, o conteúdo/programa no instante t é diferente de qualquer outro instante.

2

Versão Forte

∀t₁ ∀t₂ (t₁ ≠ t₂ → P(t₁) ≠ P(t₂))

Para quaisquer dois instantes distintos, o conteúdo é sempre diferente.

3

Versão Modal

□ ¬Igual

É necessariamente verdade que nunca é igual (lógica modal).

4

Versão Curta & Impactante

∀t. ¬Estático(t)

Em todo instante, não é estático / nunca é o mesmo.
