International Postcode System Using Cubic Meters

The Cubic Postcode is a revolutionary proposal for a universal geolocation system that assigns a unique identifier to every cubic meter on Earth. By dividing the planet into a grid of cubes with edges of one meter, and organizing them within a larger cube measuring 21 million meters on each side, this system offers a simple, unambiguous, and standardized method for pinpointing any location on the planet.

Alignment of the cubic grid with the Earth's axes

Technical Overview

The cubic grid is centered at the Earth's center of mass and aligned along the Earth's rotational axis (North-South), the prime meridian, and the equatorial plane. The grid consists of 21 million cubes along each axis, totaling a staggering 9,261,000,000,000,000,000,000 (21 million cubed) individual cubes. Each cube is uniquely numbered from the center outward in a systematic spiral pattern, ensuring that every cubic meter of space has a distinct identifier.

The numbering starts from the center cube and expands outward layer by layer. Odd-numbered layers extend along the positive X-axis, while even-numbered layers extend along the negative X-axis. This methodical approach ensures a uniform distribution of cube identifiers throughout the three-dimensional space.

How It Works

Imagine the Earth enveloped in a giant cube divided into 21 million smaller cubes along each axis. Each smaller cube measures one cubic meter and is assigned a unique 22-digit identifier. This identifier corresponds to its position within the grid, allowing for precise location referencing.

The cubes are numbered starting from the center of the Earth and moving outward in a spiral pattern. This method ensures an orderly and logical distribution of identifiers, facilitating easy calculation and retrieval of location data.

Converting Cube Number to Coordinates


# Converts a cube number to ECEF (Earth-Centered, Earth-Fixed) coordinates
# Author: Mr. Cubic Postcode, BEng (Hons)
# License: Creative Commons Attribution 4.0 International

cube = 3141592653589793238462  # Example cube number inspired by Pi
layer_size = 441000000000000  # Number of cubes in one layer

cube_projection = cube % layer_size
n_layer = cube // layer_size + 1

# Calculate square layer
n_square = int((((cube_projection - 1) ** 0.5) / 2) + 1)

# Calculate positions
right_top = (2 * (n_square - 1)) ** 2 + 2 * n_square
right_bottom = right_top + 2 * n_square - 1
left_bottom = right_bottom + 2 * n_square - 1

# Calculate Y coordinate
if right_top <= cube_projection <= right_bottom:
    Y_meters = n_square
elif cube_projection >= left_bottom:
    Y_meters = -n_square + 1
elif cube_projection < right_top:
    Y_meters = cube_projection - right_top + n_square
else:
    Y_meters = right_bottom - cube_projection + n_square
Y_meters -= 0.5

# Calculate Z coordinate
if cube_projection <= right_top:
    Z_meters = n_square - 1
elif cube_projection >= right_bottom and cube_projection <= left_bottom:
    Z_meters = -n_square
elif cube_projection < right_bottom:
    Z_meters = right_top - cube_projection + n_square - 1
else:
    Z_meters = cube_projection - left_bottom - n_square
Z_meters += 0.5

# Calculate X coordinate based on layer
if n_layer % 2 != 0:
    X_meters = n_layer / 2.0
else:
    X_meters = -(n_layer - 1) / 2.0

print('X_meters:', X_meters)
print('Y_meters:', Y_meters)
print('Z_meters:', Z_meters)

Converting Coordinates to Cube Number


# Converts ECEF coordinates to a cube number
# Author: Mr. Cubic Postcode, BEng (Hons)
# License: Creative Commons Attribution 4.0 International

import sys

def coord_round(x):
    return int(x) if x >= 0 else int(x) - 1

# Provide X, Y, Z coordinates
if len(sys.argv) == 4:
    X = float(sys.argv[1])
    Y = float(sys.argv[2])
    Z = float(sys.argv[3])
else:
    # Example coordinates corresponding to the cube number above
    X = 3561896.5
    Y = -8402969.5
    Z = -9767161.5

# Round coordinates to integers
x, y, z = (coord_round(coord) for coord in (X, Y, Z))

# Calculate square number
square_number = max(y + 1, z + 1, -y, -z)

# Calculate corner cube numbers
left_top = (2 * (square_number - 1)) ** 2 + 1
right_top = left_top + 2 * square_number - 1
right_bottom = right_top + 2 * square_number - 1
left_bottom = right_bottom + 2 * square_number - 1

# Determine cube projection number
if square_number == -y:
    if square_number == z + 1:
        cube_p = left_top
    else:
        cube_p = z + left_bottom + square_number
elif square_number == y + 1:
    cube_p = right_top + square_number - 1 - z
elif square_number == z + 1:
    cube_p = right_top - square_number - y
else:
    cube_p = right_bottom + square_number - y

# Calculate layer number
if x >= 0:
    layer_number = x * 2
else:
    layer_number = -x * 2 - 1

# Combine to get cube number
code = layer_number * 441000000000000 + cube_p

print('Cube Number:', code)
